## How to Upgrade

Follow these steps to update your Vivek Dinanath Documentation instance.

<Steps>
  <Step title="Check Current Version" icon="info">
    Run the version check command in your terminal.

    ```bash
    npx vivedk-docs --version
    ```
  </Step>
  <Step title="Install Latest" icon="download">
    Use your preferred package manager.

    <CodeGroup tabs="npm,yarn,pnpm">
      ```bash
      npm install vivedk-docs@latest
      ```
      ```bash
      yarn add vivedk-docs@latest
      ```
      ```bash
      pnpm add vivedk-docs@latest
      ```
    </CodeGroup>
  </Step>
  <Step title="Restart Server" icon="refresh-cw">
    Rebuild and restart your documentation server.

    ```bash
    npm run docs:build
    npm run docs:dev
    ```
  </Step>
</Steps>

<Callout kind="tip">
  Backup your `docs/` directory before upgrading to avoid data loss.
</Callout>
