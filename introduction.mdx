## Overview

Vivek Dinanath Documentation provides a centralized space for you to organize, manage, and share all your project documentation. You can create structured pages, embed code examples, and collaborate seamlessly with your team. This platform supports rich MDX content, making your docs interactive and professional.

### Integrate with Your Workflow

Embed documentation in your apps using simple API calls.

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  const response = await fetch('https://api.vivekdocs.com/v1/pages', {
    headers: { Authorization: `Bearer ${YOUR_API_KEY}` }
  });
  const pages = await response.json();
  console.log(pages);
  ```
  ```python
  import requests
  headers = {'Authorization': f'Bearer {YOUR_API_KEY}'}
  response = requests.get('https://api.vivekdocs.com/v1/pages', headers=headers)
  pages = response.json()
  print(pages)
  ```
</CodeGroup>
